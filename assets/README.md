# Assets Directory

Place your screenshot image here as `screenshot.png`.

## Recommended Image Specifications:
- **Format**: PNG or JPG
- **Size**: 1200x800px or similar aspect ratio (3:2)
- **File name**: `screenshot.png` or `screenshot.jpg`

The image will be displayed on:
- pub.dev package page
- GitHub repository README

## How to add your screenshot:

1. Take a screenshot of your floor plan editor in action
2. Save it as `screenshot.png` in this `assets/` directory
3. Commit and push to GitHub
4. The image will automatically appear in your README
